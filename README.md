# Bank Management System

A simple Bank Management System developed using Python and Object-Oriented Programming (OOP).

## Features

* Create Bank Accounts
* Deposit Money
* Withdraw Money
* Transfer Money
* PIN Authentication
* Check Balance
* Transaction History
* Savings Account Interest
* Current Account Overdraft

## Technologies Used

* Python
* OOP (Object-Oriented Programming)

## Concepts Covered

* Classes and Objects
* Encapsulation
* Inheritance
* Method Overriding

## Run the Project

```bash
python Banking.py
```
